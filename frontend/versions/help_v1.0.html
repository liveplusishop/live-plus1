📌 一、FB 授權設定

在使用系統之前，需要先綁定你的 Facebook 粉絲專頁

需要的資料：

欄位說明
粉絲專頁 IDFB 粉絲團的數字 ID（15-16位數）
Access TokenFB 長期權杖（有效期幾個月）

如何取得 Access Token：

1

進入 FB Developer

前往 developers.facebook.com
2

選擇你的粉絲專頁 App

或建立一個新的 App
3

取得長期權杖

使用「取得權杖」→「取得頁面存取權杖」
4

複製貼上

將取得的 Token 複製到系統設定中

步驟 1：安裝 Python（必須）

Python 是程式的執行環境，必須先安裝。

❓ 六、常見問題

Q：Access Token 過期怎麼辦？
A：需要到 FB Developer 重新取得新的 Token，然後更新系統設定
Q：觀眾不會用怎麼辦？
A：把展示頁（demo.html）分享給他們看操作流程
Q：可以綁定多個 FB 粉絲團嗎？
A：目前一個帳號綁定一個粉絲團
Q：留言識別不出來怎麼辦？
A：檢查 FB 授權是否正確，或手動新增關鍵字（如「+1」、「要」）
Q：自動列印需要什麼設備？
A：需要一台 Windows 電腦（10/11）和印表機
Q：可以用手機自動列印嗎？
A：手機無法直接控制印表機，需要電腦。小程式在辦公室電腦背景執行即可
Q：如何取得自動列印程式？
A：聯繫 LINE 客服 或 系統管理員 取得下載連結
