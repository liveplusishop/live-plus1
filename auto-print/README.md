# 直播加1 - 自動列印小程式

## 📌 功能說明

這個小程式可以自動偵測直播加1系統的新訂單，並自動列印出貨單。

### 主要功能

- 🔔 **自動監控** - 每 N 秒檢查一次新訂單
- 🖨️ **自動列印** - 有新訂單時自動列印出貨單
- 📊 **運行記錄** - 顯示已列印數量、運行時間
- 💬 **系統通知** - 新訂單時顯示 Windows 通知

---

## 🚀 使用方式

### 方式一：直接啟動（推薦）

1. 雙擊 `啟動.bat`
2. 設定 API 網址（預設是 Railway 雲端）
3. 選擇印表機
4. 點「開始監控」

### 方式二：命令列啟動

```bash
cd auto-print
python auto_print.py
```

---

## ⚙️ 設定說明

| 設定 | 說明 |
|------|------|
| API 網址 | 直播加1的 API 地址 |
| 檢查間隔 | 每幾秒檢查一次（預設 5 秒）|
| 印表機 | 選擇要使用的印表機 |
| 自動列印 | 勾選後有新訂單自動列印 |

## ⚠️ 常見問題

### Q: 為什麼需要點一下才能列印？

A: 因為瀏覽器安全限制，無法直接送到印表機。
   建議將 Chrome/Edge 設為「自動列印到預設印表機」：
   設定 → 隱私與安全性 → 網站設定 → 額外內容設定 → 自動列印

### Q: 可以不要彈出瀏覽器嗎？

A: 目前需要瀏覽器來處理列印。未來可支援直接列印（需額外設定）。
