# 直播加1系統 - 填寫範例指南

## 📌 一、FB 授權設定

### 需要的資料：
| 欄位 | 範例填寫 | 說明 |
|------|----------|------|
| 粉絲專頁 ID | `1234567890123456` | FB 粉絲團的數字 ID |
| Access Token | `EAAxxxxxxxxxxxx...` | FB 長期權杖（有效期幾個月） |

### 如何取得：
1. 進入 [FB Developer](https://developers.facebook.com/)
2. 選擇你的粉絲專頁 App
3. 取得長期 Access Token
4. 複製貼上即可

## 📌 四、如何讓觀眾下單

### 直播時的操作：
1. 在 FB 直播中說：「要購買的粉絲，在底下留言『+1』」
2. 系統自動抓取留言 `+1`
3. 自動發送「買家填單連結」給觀眾
4. 觀眾點連結填寫收件資料
5. 你在「訂單管理」查看所有訂單

### 觀眾看到的買家表單：
```
直播加1 收件表單
請填寫收件資料，完成訂購

地址：台北市信義區忠孝東路123號
備註：
```

## 📌 六、常見問題

**Q：Access Token 過期怎麼辦？**
A：需要到 FB Developer 重新取得新的 Token

**Q：觀眾不會用怎麼辦？**
A：把 demo.html 展示頁分享給他們看操作流程

**Q：可以綁定多個 FB 粉絲團嗎？**
A：目前一個帳號綁定一個粉絲團
